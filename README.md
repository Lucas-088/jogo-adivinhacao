# Jogo da Adivinhação

Um jogo simples em Python onde o jogador tenta adivinhar um número secreto entre 1 e 100.

## Descrição

O programa gera um número aleatório entre 1 e 100 e o jogador deve tentar adivinhar.  
A cada tentativa, o jogo informa se o número secreto é maior ou menor que o palpite.  

## Como executar

1. Certifique-se de ter o **Python 3.13.6** instalado.
2. Baixe ou clone este repositório.
3. No terminal, navegue até a pasta do projeto e execute:
python jogo_adivinhacao.py
